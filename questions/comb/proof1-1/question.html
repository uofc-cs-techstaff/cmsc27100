A DNA strand can be viewed as a string over the alphabet $\{A,C,G,T\}$ of DNA bases. While DNA is a double-stranded structure, it is often enough to consider one of the two strands as a string because the other strand is uniquely determined by Watson-Crick base pairs. We define the Watson-Crick complement $\theta(w)$ of a string $w$ inductively by:

  • if $w = \varepsilon$ is empty, then $\theta(\varepsilon) = \varepsilon$,
  • if $w = a \in \{A,C,G,T\}$ is a DNA base, then $\theta(A) = T, \theta(C) = G, \theta(G) = C, \theta(T) = A$,
  • if $w = ax$ for a DNA strand $x \in \{A,C,G,T\}^*$ and a DNA base $a \in \{A,C,G,T\}$, then $\theta(w) = \theta(x)\theta(a)$.

In other words, the Watson-Crick function $\theta$ swaps each letter representing a base to the corresponding complementary base and reverses the string. For example, $$\theta(ATTACG) = CGTAAT.$$ One kind of interesting DNA strand is a string that is equal to its own Watson-Crick complement. That is, $w = \theta(w)$. Such strings are called Watson-Crick palindromes or $\theta$-palindromes.

How many DNA strings of length $n$ are Watson-Crick palindromes if $n$ is odd?

Hidden.

 If $n$ is odd, then $n = 2k+1$ for some integer $k$. Since $n = 2k+1$, we can factor our string into $w = uav$, where $u$ and $v$ are strings of length $k$ and $a$ is a single DNA base. Since $w = \theta(w)$, we have $\theta(w) = \theta(uav) = \theta(v)\theta(a)\theta(u)$. Therefore, we have $u = \theta(v)$, $v = \theta(u)$, and $a = \theta(a)$. But there is no single DNA base $a$ such that $a = \theta(a)$. Therefore, there are no Watson-Crick palindromes of odd length. Then $v$ is uniquely determined by our choice of $u$. Since $u$ is of length $k$, there are $4^k$ possible choices for $u$. Since there are four choices for $a$, there are $4^{k+1}$ Watson-Crick palindromes of length $n = 2k+1$. Since $u$ is of length $k$, there are $\binom k 4$ possible choices for $u$. Since there are four choices for $a$, there are $4 \cdot \binom k 4$ Watson-Crick palindromes of length $n = 2k+1$. Since $u$ is of length $k$, there are $P(k,4)$ possible choices for $u$. Since there are four choices for $a$, there are $4 \cdot P(k,4)$ Watson-Crick palindromes of length $n = 2k+1$.

A DNA strand can be viewed as a string over the alphabet $\{A,C,G,T\}$ of DNA bases. While DNA is a double-stranded structure, it is often enough to consider one of the two strands as a string because the other strand is uniquely determined by Watson-Crick base pairs. We define the Watson-Crick complement $\theta(w)$ of a string $w$ inductively by:

  • if $w = \varepsilon$ is empty, then $\theta(\varepsilon) = \varepsilon$,
  • if $w = a \in \{A,C,G,T\}$ is a DNA base, then $\theta(A) = T, \theta(C) = G, \theta(G) = C, \theta(T) = A$,
  • if $w = ax$ for a DNA strand $x \in \{A,C,G,T\}^*$ and a DNA base $a \in \{A,C,G,T\}$, then $\theta(w) = \theta(x)\theta(a)$.

In other words, the Watson-Crick function $\theta$ swaps each letter representing a base to the corresponding complementary base and reverses the string. For example, $$\theta(ATTACG) = CGTAAT.$$ One kind of interesting DNA strand is a string that is equal to its own Watson-Crick complement. That is, $w = \theta(w)$. Such strings are called Watson-Crick palindromes or $\theta$-palindromes.

How many DNA strings of length $n$ are Watson-Crick palindromes if $n$ is even?

Note. Not every line belongs in the proof.

Hidden.

 If $n$ is even, then $n = 2\ell$ for some integer $\ell$. Since $n = 2\ell$, we can factor our string into $w = uv$, where $u$ and $v$ are strings of length $\ell$. Since $w = \theta(w)$, we have $u = \theta(v)$ and $v = \theta(u)$. This means that the first half of $w$ determines the latter half of $w$. Since $w$ is a DNA string of length $\ell$, there are $4^\ell$ possible strings that make up the first half of $w$. Since $\ell = \frac n 2$, there are $4^{\frac n 2}$ possible strings of length $n$ such that $w = \theta(w)$ when $n$ is even. Then there are $\binom \ell 4$, where $\ell = \frac n 2$, possible strings of length $n$ such that $w = \theta(w)$ when $n$ is even. Thus, there are $2 \cdot 4^{\frac n 2}$ possible strings of length $n$ such that $w = \theta(w)$ when $n$ is even. Therefore, there are $4^{2\ell}$ possible strings of length $n$ such that $w = \theta(w)$ when $n$ is even. Then if $\ell = \frac n 2$, there are $P(\ell,4)$ possible strings of length $n$ such that $w = \theta(w)$ when $n$ is even.

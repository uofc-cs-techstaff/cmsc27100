Use a combinatorial proof to show that \[n2^{n-1} = \sum_{k=1}^n k \binom n k.\]

Hidden.

 Suppose we have a binary string (a string over $\{0,1\}$) of length $n$ with a special marker $\#$ in one of the positions. We will show two ways to count these strings. We first select a position for the marker $\#$. There are $n$ such possibilities. Then, there are $n-1$ positions left for either $0$s or $1$s to be filled. Since each of the $n-1$ positions can be filled by either $0$ or $1$, there are two choices for each position. Therefore, there are $2^{n-1}$ different ways to fill the $n-1$ positions. This gives us $n2^{n-1}$ possible binary strings of length $n$ that contain $\#$. We first choose $k$ out of $n$ positions. There are $\binom n k$ such choices. Of these $k$ choices, we choose one to be filled by $\#$ and fill the other $k-1$ positions with $0$s. The $n-k$ positions that were not chosen are filled with $1$s. This gives a string a length $n$ and there are $k \binom n k$ such strings. For all possible strings of length $n$, we sum over all possible $k$. The range of $k$ is from $1$ to $n$, since any string must contain the marker $\#$. This gives us \[\sum_{k=1}^n k \binom n k\] possible binary strings of length $n$ that contain $\#$. Therefore, \[n2^{n-1} = \sum_{k=1}^n k \binom n k.\]

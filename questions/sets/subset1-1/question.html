Prove that if $A \subseteq B$, then $A \cap \overline B = \emptyset$.

Hidden.

 Assume $A \subseteq B$. Suppose that $A \cap \overline B$ is not empty. Then there is an element $x \in A \cap \overline B$. By definition of intersection and complement, $x \in A$ and $x \not \in B$. Since $A \subseteq B$, if $x \in A$, then $x \in B$. This is a contradiction. Therefore, $A \cap \overline B = \emptyset$.

Prove that if $A \cap \overline B = \emptyset$, then $A \subseteq B$.

Hidden.

 Assume $A \cap \overline B = \emptyset$. Suppose that $A \not \subseteq B$. By definition of negation of subset, this means there is an element $x$ such that $x \in A$ and $x \not \in B$. Then by definition of intersection and complement, there is an element $x \in A \cap \overline B$. This is a contradiction. Therefore, $A \subseteq B$.

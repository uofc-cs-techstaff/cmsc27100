Prove that $\Pr(\bar A ) = 1 - \Pr(A)$.

Hidden.

 By definition of probability, $\Pr(\bar A) = \frac{|\bar A|}{|\Omega|}$. By definition of complement, $\bar A = \Omega \setminus A$. Then $\frac{|\bar A|}{|\Omega|} = \frac{|\Omega \setminus A|}{|\Omega|}$. By definition of set subtraction, $\frac{|\Omega \setminus A|}{|\Omega|} = \frac{|\Omega| - |A|}{|\Omega|}$. Then $\frac{|\Omega| - |A|}{|\Omega|} = \frac{|\Omega|}{|\Omega|} - \frac{|A|}{|\Omega|} = 1 - \frac{|A|}{|\Omega|}$. By definition of probability, $\frac{|A|}{|\Omega|} = \Pr(A)$. Therefore, $1-\frac{|A|}{|\Omega|} = 1-\Pr(A)$. Therefore, $\Pr(\bar A) = 1-\Pr(A)$.

Let $P$ and $Q$ be predicates. Prove that if $\neg P(y)$ is true, then $\exists x (P(x) \rightarrow Q(y))$.

Suppose $\neg P(y)$ is true. Assume $P(y)$ is true. This is a contradiction. Therefore, we can conclude $Q(y)$. Thus, we can conclude that $P(y) \rightarrow Q(y)$. Therefore, $\exists x (P(x) \rightarrow Q(y))$.

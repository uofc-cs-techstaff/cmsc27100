Let $p$, $q$, and $r$ be propositions. Prove that if $(p \wedge q) \rightarrow r$ then $p \rightarrow (q \rightarrow r)$.

Suppose $(p \wedge q) \rightarrow r$ is true. Assume $p$ is true. Assume $q$ is true. Since $p$ is true and $q$ is true, we can conclude that $p \land q$ is true. Since $p \land q$ is true we can conclude $r$ is true. Therefore, we can conclude $q \rightarrow r$. Therefore, we can conclude $p \rightarrow (q \rightarrow r)$.
